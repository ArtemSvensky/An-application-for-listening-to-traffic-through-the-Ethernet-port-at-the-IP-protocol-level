Приложение для прослушивания трафика через Ethernet порт на уровне протокола IP

Важно! Могут быть проблемы с работой приложения, поэтому необходимо будет запустить его через консоль от имени администратора.

Чтобы проверить корректную работу приложения необходимо открыть в консоле директорию с проектом и с помощью скрипта "sniffer.exe 192.168.1.200 data.log" запустить его 

Консольное приложение (sniffer.exe), прослушивающие трафик, проходящий через Ethernet порт на уровне протокола IP, и записывающие информацию о прошедшем трафике в указанный файл.

Параметры командной строки:

Sniffer.exe <IP> <файл>

При старте приложение настраивает сырой сокет на прослушивание входящего/исходящего трафика. В файл записываются только распарсенные заголовки каждого IP пакета. Если IP пакет содержит UDP или TCP пакет, их заголовки также распарсиваются и сохраняются. Формат записи в файл может быть выбран на усмотрение разработчика, но записи должны быть читаемыми в простых текстовых редакторах, таких как блокнот.
